Project: Task Management Dashboard

Build a task management application that demonstrates your React expertise, state management, and attention to detail.

Core Features Required

Task CRUD Operations
Display tasks with: id, title, description, status (todo, in-progress, completed), priority (low, medium, high), and due date
Add new tasks via form
Edit existing tasks
Delete tasks
Mark tasks as complete/incomplete
Filtering & Sorting
Filter by status and priority
Sort by due date or priority
Search tasks by title
Task Form
Validate that title is required
Validate that due date is in the future
Show error messages for invalid inputs
Pre-populate form in edit mode
Clear form after successful submission
User Interface
Clean, responsive design
Card-based or table layout for tasks
Visual indicators for priority levels (colors/badges)
Display task counts for each status
Search functionality
